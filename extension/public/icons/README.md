# Icons

Place your extension icons here:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

For now, you can use placeholder images or create simple icons with the ✨ emoji.

To create icons quickly:
1. Use an online tool like https://favicon.io/
2. Or create simple PNG files with your logo
3. Or use emoji-to-png converters

The extension will work without icons, but Chrome will show a default placeholder.
